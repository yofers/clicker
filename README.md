# Clicker点击器 (连点器 & 长按器)

Clicker 是一个基于 [Wails](https://wails.io/) 构建的跨平台自动点击/长按工具，支持 macOS 和 Windows。它拥有现代化的 Glassmorphism UI 设计，提供灵活的鼠标和键盘模拟功能。

## ✨ 主要功能

- **双模式支持**：
  - **连点器 (Clicker)**：支持自定义间隔的连续点击或键盘按键模拟。
  - **长按器 (Presser)**：支持“按住不放”模式，模拟持续按压鼠标或键盘。
- **多种点击类型**：
  - 单击 (Single Click)
  - 双击 (Double Click)
  - 长按 (Hold/Long Press)
- **全方位输入模拟**：
  - **鼠标**：支持左键、右键、中键以及侧键 (前进/后退)。
  - **键盘**：支持绝大多数常用按键 (F1-F12, 字母, 数字, 功能键等)。
- **全局快捷键**：
  - 使用 `Ctrl+F8` (macOS 上可能需要 `Fn+Ctrl+F8`) 随时启动或停止。
- **现代化 UI**：
  - 精美的磨砂玻璃效果 (Glassmorphism)。
  - 自定义背景支持。
  - 响应式交互动画。

## 🛠️ 安装与构建

### 前置要求

- [Go](https://go.dev/) (1.18+)
- [Wails](https://wails.io/) CLI
- Node.js & npm

### 构建步骤

1. 克隆项目：
   ```bash
   git clone https://github.com/yofers/clicker.git
   cd clicker
   ```

2. 安装依赖：
   ```bash
   # 进入前端目录安装依赖
   cd frontend
   npm install
   cd ..
   ```

3. 运行开发模式：
   ```bash
   wails dev
   ```

4. 构建生产版本：
   ```bash
   wails build
   ```
   构建产物将位于 `build/bin` 目录下。

## 📖 使用指南

### 1. 启动应用
打开应用后，你将看到简洁的主界面。点击左上角的切换按钮可以在“连点器”和“长按器”模式之间切换。

### 2. 连点器模式 (Clicker)
适用于需要高频重复点击或按键的场景。
- **间隔**：设置每次点击之间的等待时间 (毫秒)。
- **模式**：选择模拟“鼠标”或“键盘”。
- **按键**：选择具体的鼠标按键 (如左键) 或键盘按键 (如 Space)。
- **点击模式**：
  - **单击**：标准的点击。
  - **双击**：每次触发执行两次点击。
  - **长按**：每次触发按住指定时长 (需设置长按时长)。

### 3. 长按器模式 (Presser)
适用于需要持续按住某个键不放的场景 (如游戏中的奔跑、持续施法)。
- **操作**：启动后，程序会按下指定的键并保持按住状态，直到你再次按下快捷键停止。

### 4. 快捷键
- **启动/停止**：`Ctrl + F8`
- **macOS 注意事项**：如果你的 F8 键默认为媒体键，请使用 `Ctrl + Fn + F8`。

## ⚠️ macOS 权限说明

由于应用需要模拟系统级的输入事件，在 macOS 上首次运行时需要授予**辅助功能权限 (Accessibility)**。

1. 点击“启动”或按下快捷键时，系统会弹出权限请求提示。
2. 点击“打开系统设置”。
3. 在“隐私与安全性” -> “辅助功能”列表中，找到 `Clicker` (或终端应用) 并开启开关。
4. 如果授权后仍无效，尝试重启应用。
